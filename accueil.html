Une petite présentation

Chère visiteur(euse), Bonjour !

Bienvenue sur mon site. Il à pour objectif de me présenter et de vous montrer mes capacités en tant que Développeur Web.

Je m'appelle Geoffrey DUBOIS, j'ai 28 ans, je suis originaire de l'Ile de France et plus précisément du Val d'Oise.

Vous trouverez sur ce site des informations sur ma vie personnelle ainsi que professionnelle afin d'avoir une première idée de qui je suis.

Avant toutes choses, je vous précise que ce site à était concu et développer avec les technologies Html5/Css3 pour le coté intégration. Pour toutes les animations que vous allez voir sur ce site, j'ai souhaité utilisé Jquery pour un rendu plus agreable que des animations faites en Css.

Ce site est divisé en 5 parties. Vous vous trouvez actuellement sur la page d'accueil qui à pour but de vous présenter brievement le site et mes compétences. Mes compétences ainsi que mon cursus scolaire et professionel sont développer dans la partie CV. Je vous développerez plus mon parcours personnel dans l'onglet Mon Parcours. Pour vous montrer mes compétences acquise dans le domaine du développement, j'ai mis en pratique mes techniques dans Porfolio. Enfin, si vous voulez me contacter directement par mail, j'ai fait un formulaire de contact que vous trouverez Ici.

La prochaie évolution de ce site sera de modifier le code css pour avoir un site en Responsive Design. Pour cela, je compte utiliser la technologie Media Queries

J'espère que mon site vous plaira et qu'ont aura l'occasion de ce rencontrer pour un entretien. Sur ce, je vous souhaite une très bonne visite !

Dubois Geoffrey - Développeur Web - 2015
